####定义Bootstrap导航栏

```
from flask_bootstrap import Bootstrap
from flask_nav import Nav
from flask_nav.elements import *

Bootstrap(app)
nav = Nav()

nav.register_element('top', Navbar('Flask',
					View('Home', 'index'),
					View('Upload', 'upload')
					))

nav.init_app(app)
```

使用的时候，在模版调用```{{ nav.top.render() }}```
